# Statistical Learning & Regression Modeling

Welcome to my machine learning and statistical modeling repository! This repository contains my coursework, projects, and notes from an 11-week deep dive into regression analysis, classification, and statistical learning models.

## 📖 Overview

This repository tracks my progress through various predictive modeling techniques, starting from foundational simple linear regression and advancing through generalized linear models, resampling methods, and non-linear modeling. 

## 📂 Repository Structure

The curriculum is organized week-by-week, alongside dedicated folders for supplementary materials and applied projects.

### Coursework by Week
* **[Week 1 - Simple Linear Regression](./Week%201%20-%20Simple%20Linear%20Regression)**: Fundamentals of predicting a quantitative response using a single feature variable.
* **[Week 2 - Multiple Linear Regression](./Week%202%20-%20Multiple%20Linear%20Regression)**: Extending linear regression to accommodate multiple predictors.
* **[Week 3 - Other Consideration](./Week%203%20-%20Other%20Consideration)**: Handling qualitative predictors, interaction terms, and potential problems in linear models.
* **[Week 4 - Why not Linear Regression](./Week%204%20-%20Why%20not%20Linear%20Regression)**: Understanding the limitations of linear regression for classification tasks.
* **[Week 5 - Logistic Regression](./Week%205%20-%20Logistic%20Regression)**: Introduction to modeling binary categorical outcomes.
* **[Week 6 - Multinomial](./Week%206%20-%20Multinomial)**: Extending logistic regression to multi-class classification problems.
* **[Week 7 - Linear Discriminate Analysis](./Week%207%20-%20Linear%20Discriminate%20Analysis)**: Using LDA for classification and dimensionality reduction.
* **[Week 8 - Generalized Linear Models](./Week%208%20-%20Generalized%20Linear%20Models)**: Exploring the GLM framework, including Poisson regression and other link functions.
* **[Week 9 - Resampling Methods](./Week%209%20-%20Resampling%20Methods)**: Cross-validation and the bootstrap for model evaluation and assessing variability.
* **[Week 10 - Model Selection](./Week%2010%20-%20Model%20Selection)**: Subset selection, shrinkage methods (Ridge/Lasso), and dimension reduction techniques.
* **[Week 11 - Moving Beyond Linearity](./Week%2011%20-%20Moving%20Beyond%20Linearity)**: Polynomial regression, step functions, splines, and Generalized Additive Models (GAMs).
